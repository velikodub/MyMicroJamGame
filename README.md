# MyMicroJamGame
A project developed for Micro Jam 053.
Developed for **Micro Jam 053** **Theme:** Isolation

## Tech Stack
- **Engine:** Unity 2022.3
- **Language:** C#
- **Style:** 2D Minimalist Platformer

## Concept
A short narrative-driven platformer about robot exploring a long-forgoten Facility.
The player is "isolated" from the world by a lack of vision, replying on acoustic pulses (echolocation) to navigate

## How to Play
- [Space] - Emit Echo Pulse
- [WASD/Arrows] - Movement
- [E] - Interact with terminals
